> **AION** is my flagship project: a **time-series harness** — a control layer that
> connects task specification, runtime execution, and result assessment into one
> stable process for next-generation time-series workloads. Built on top of
> [OpenCode](https://github.com/anomalyco/opencode), with **4 layers** (Task /
> Workspace / Execution / Review), **6 agents**, **8 protocols**, and **17 skills**.
